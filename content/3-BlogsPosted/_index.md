This section lists and introduces the blogs I have posted to the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj) community.

###  [Blog 1 - KIRO: AWS AI Coding Assistant and the AI-Assisted Development Lifecycle](3.1-Blog1/)
An introduction to Kiro, AWS's AI IDE / Coding Assistant, viewed not merely as a "code generator" but as a tool that guides developers through a more structured, Spec-Driven Development workflow. The blog also connects Kiro to the AI-assisted development lifecycle described in the updated AWS Well-Architected Machine Learning Lens.

###  [Blog 2 - Running a Traditional Web Application on AWS Nitro Enclaves Without Modifying Source Code](3.2-Blog2/)
An overview of AWS Nitro Enclaves and a practical approach for moving existing HTTP/HTTPS web applications into an isolated enclave using SOCAT as an HTTP-to-VSOCK proxy layer, so the application can run almost unchanged while keeping the strong isolation Nitro Enclaves provides.

###  [Blog 3 - AWS Pricing Calculator: Estimating Costs Before Deployment](3.3-Blog3/)
A short, hands-on guide to the AWS Pricing Calculator, a free tool for estimating monthly costs before provisioning any resources, walking through the workflow of creating an estimate for a basic EC2 workload.
